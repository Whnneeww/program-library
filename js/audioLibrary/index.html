AudioLibrary 使用ガイド（バージョン 0.1.0）

AudioLibraryは、音声ファイルを処理し、HAX形式のデータに変換するための便利なツールです。このライブラリの新機能について説明します。

導入方法

AudioLibraryを使用するには、以下のスクリプトをHTMLに追加します：

<script src="https://program-library.i-i.f5.si/js/audioLibrary/0.1.0.js"></script>

基本的な使い方

1. オーディオプロセッサの作成

まず、AudioProcessorクラスのインスタンスを生成します。

const audioProcessor = new AudioProcessor();

2. 音声ファイルの処理

音声ファイルを選択して処理するためのイベントリスナーを追加します。


document.getElementById('audioFileInput').addEventListener('change', async (event) => {
    const file = event.target.files[0];
    if (file) {
        await audioProcessor.processAudioFile(file);
        document.getElementById('playButton').disabled = false; // 再生ボタンを有効化
    }
});

3. HAXデータの再生

生成されたHAXデータを再生するためのボタンを設定します。


document.getElementById('playButton').addEventListener('click', () => {
    audioProcessor.playHAXData(audioProcessor.haxData);
});

4. 完全なサンプルコード

以下は、AudioLibraryの使用方法を示す完全なサンプルコードです。


<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>AudioLibrary Demo</title>
    <script src="https://program-library.i-i.f5.si/js/audioLibrary/0.1.0.js"></script>
</head>
<body>
    <h1>AudioLibrary Demo</h1>
    <input type="file" id="audioFileInput" accept="audio/mp3" />
    <button id="playButton" disabled>HAXデータを再生</button>
    
    <script>
        const audioProcessor = new AudioProcessor();

        document.getElementById('audioFileInput').addEventListener('change', async (event) => {
            const file = event.target.files[0];
            if (file) {
                await audioProcessor.processAudioFile(file);
                document.getElementById('playButton').disabled = false; // 再生ボタンを有効化
            }
        });

        document.getElementById('playButton').addEventListener('click', () => {
            audioProcessor.playHAXData(audioProcessor.haxData);
        });
    </script>
</body>
</html>

注意事項

ライブラリのバージョンによっては、APIの仕様が変わる可能性があります。最新のドキュメントを確認してください。

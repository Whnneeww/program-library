Whnライブラリ 使用ガイド

Whnライブラリは、ファイルのアーカイブ作成を簡単に行うための便利なツールです。以下では、このライブラリの基本的な使い方を説明します。

導入方法

Whnライブラリを使用するには、以下のスクリプトをHTMLに追加します：

<script src="https://program-library.i-i.f5.si/js/whn/0.0.2-a.js"></script>

基本的な使い方

1. アーカイブオブジェクトの作成

アーカイブを作成するために、まずWhnneewwArchiveクラスのインスタンスを生成します。

const archive = new WhnneewwArchive();

2. ファイルの追加

ファイルをアーカイブに追加するには、addFileメソッドを使用します。

archive.addFile(file); // 'file'はFileオブジェクト

3. アーカイブの保存

すべてのファイルを追加したら、saveメソッドを使ってアーカイブを保存します。

archive.save('myArchive.zip');

4. 完全なサンプルコード

以下は、ユーザーが選択したファイルをアーカイブにまとめてダウンロードするサンプルコードです。


    <!DOCTYPE html>
    <html lang="ja">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>ファイルアーカイブツール</title>
        <script src="https://program-library.i-i.f5.si/js/whn/0.0.2-a.js"></script>
    </head>
    <body>
        <h1>ファイルアーカイブツール</h1>
        <input type="file" id="fileInput" multiple />
        <button id="createArchive">アーカイブ作成</button>

        <script>
            const fileInput = document.getElementById('fileInput');
            const createArchiveButton = document.getElementById('createArchive');

            let filesToArchive = [];

            fileInput.addEventListener('change', (event) => {
                filesToArchive = [...event.target.files];
            });

            createArchiveButton.addEventListener('click', () => {
                const archive = new WhnneewwArchive();
                filesToArchive.forEach(file => {
                    archive.addFile(file);
                });
                archive.save('myArchive.zip');
                alert('アーカイブが作成されました！');
            });
        </script>
    </body>
    </html>

注意事項

ライブラリのバージョンによっては、APIの仕様が変わる可能性があります。最新のドキュメントを確認してください。

まとめ

Whnライブラリを利用することで、ファイルのアーカイブ作成がシンプルになります。問題があれば、いつでも質問してくださいね！

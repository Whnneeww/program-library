Whnライブラリ 使用ガイド（新バージョン）

Whnライブラリは、ファイルとフォルダーの管理を簡単に行うための便利なツールです。このライブラリの新機能について説明します。

導入方法

Whnライブラリを使用するには、以下のスクリプトをHTMLに追加します：

<script src="https://program-library.i-i.f5.si/js/whn/2.0.1.js"></script>

基本的な使い方

1. アイテムの作成

新しいアイテムを作成するために、WhneewwItemクラスのインスタンスを生成します。

const item = new WhneewwItem("アイテム名", "タイプ");

2. アーカイブの作成

アイテムの配列からアーカイブを作成するには、以下のようにします。

const blob = WhneewwArchive.create([item1, item2]);

3. アーカイブの抽出

アーカイブを抽出するには、extractメソッドを使用します。

WhneewwArchive.extract(blob).then(items => { /* アイテムを処理する */ });

4. フォルダの操作

フォルダの追加、削除、名前変更が簡単にできます。


const rootFolder = new WhneewwItem('Root', 'folder');
WhneewwArchive.addFolder(rootFolder, 'NewFolder');
WhneewwArchive.renameFolder(rootFolder, 'NewFolder', 'RenamedFolder');
WhneewwArchive.deleteFolder(rootFolder, 'RenamedFolder');

5. 完全なサンプルコード

以下は新バージョンの機能を活用したサンプルコードです。


const rootFolder = new WhneewwItem('Root', 'folder');
const addFolderResult = WhneewwArchive.addFolder(rootFolder, 'NewFolder');
console.log(addFolderResult.success ? 'フォルダー追加成功' : addFolderResult.message);
const renameFolderResult = WhneewwArchive.renameFolder(rootFolder, 'NewFolder', 'RenamedFolder');
console.log(renameFolderResult.success ? 'フォルダー名称変更成功' : renameFolderResult.message);
const deleteFolderResult = WhneewwArchive.deleteFolder(rootFolder, 'RenamedFolder');
console.log(deleteFolderResult.success ? 'フォルダー削除成功' : deleteFolderResult.message);
console.log('ルートフォルダーの内容:', rootFolder.children);

注意事項

ライブラリのバージョンによっては、APIの仕様が変わる可能性があります。最新のドキュメントを確認してください。
